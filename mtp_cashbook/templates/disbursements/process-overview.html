{% trans 'How to use the new digital disbursement service' %}

{% trans 'When a prisoner wants to send money out of prison (eg, to relatives or charities), use this new disbursement service to digitally process the payment request.' %} {% trans 'You can only use this service to process bank transfers and cheques.' %} {% trans 'It works alongside the existing service of sending money out by post.' %} {% trans 'This service is free - there are no postage or processing fees' %}

{% trans 'To start using this service, go to ‘manage prisoner money’, use your digital cashbook login and click on ‘digital disbursements’' %}

{% trans 'How it works' %}

1. {% trans 'Prisoner fills in the paper from and has their identity confirmed' %}

{% url 'disbursements:paper-forms' as link %} {% blocktrans trimmed %} The wing officer gives the correct ‘disbursement form’ to the prisoner to fill in. {% endblocktrans %} {% trans 'Prisoner fills in the form and the wing officer signs it to confirm the prisoner’s identity.' %} {% trans 'It can be put into an envelope to make it secure.' %}

{% random_string as content_id %}

{% trans 'Follow normal security procedures' %}

{% trans 'When a prisoner submits a disbursement, please follow your prison’s normal security procedures.' %} {% trans 'Prison identity must be verified in person and hub staff must confirm the authorised signature on the paper form, just as before.' %} {% trans 'If forms need to be checked by security staff, before or after going to the business hub, this process should stay the same.' %}

{% random_string as content_id %}

{% trans 'Paper forms' %}

{% url 'disbursements:paper-forms' as link %} {% blocktrans trimmed %} All paper forms have been updated. {% endblocktrans %} {% trans 'Print out and use whichever of the four forms suit your prison procedures:' %}
  • {% trans 'digital disbursements (bank transfer and cheques)' %}
  • {% trans 'postal order and cash' %}
  • {% trans 'postal orders only' %}
  • {% trans 'cash only' %}

2. {% trans 'Hub staff #1 enters paper form details into the new disbursement tool' %}

{% url 'disbursements:start' as link %} {% blocktrans trimmed %} A business hub staff member enters the details from the paper form into the ‘Send payment’ section of this tool. {% endblocktrans %}

{% trans 'This tool is connected to NOMIS so it shows a prisoner’s up-to-date account information.' %} {% trans 'Money will come out of the prisoner’s private cash account.'%} {% trans 'But, as before, money can be manually moved in NOMIS by hub staff, to a private account from spends or savings, then sent out.' %}

3. {% trans 'Hub staff #2 double-checks and confirms the details on the paper form and the details in the digital tool are the same' %}

{% trans 'When the payment request has been entered into the disbursement tool, the paper form must be handed over to a colleague who has access to the tool.' %}

{% trans 'They need to log in to the tool to double-check the details entered match the paper form details.' %} ({% trans 'Login details for the digital cashbook and the disbursement tool are the same.' %} {% trans 'Ask your business hub manager if you have access problems.' %})
{% url 'disbursements:pending_list' as link %} {% blocktrans trimmed %} 'Then, if the details match, click on ‘Confirm payment’ and sign the paper form. {% endblocktrans %}

{% trans 'The confirmed requests automatically go to SSCL for processing.' %}

{% random_string as content_id %}

{% trans 'Who can’t confirm payment' %}

{% trans 'A payment can’t be confirmed by the person who:' %}
  • {% trans '‘entered’ the payment request - the process needs a different member of staff to check the details' %}
  • {% trans 'is last to make the ‘change’ or ‘edit’ to the request' %}

4. {% trans 'Payment request is automatically deducted from NOMIS then goes to SSCL which sends out payment and remittance advice' %}

{% trans 'SSCL sends the payment to the recipient by bank transfer or cheque, as requested by the prisoner.' %} {% blocktrans trimmed %} It also sends out remittance advice which is a note sent to the recipient by post or email (both free). It names the prisoner who sent the money and how much they sent. There’s a ‘recipient email’ box on the form for the prisoner to fill in, if they have it. {% endblocktrans %}

{% trans 'In 5-7 working days, either the payment is put directly into the recipient’s account by bank transfer or a cheque is received.' %} {% trans 'There’s no postage or processing charge to the prison, the prisoner or the recipient.' %}

{% trans 'If prisoner gets the recipient’s bank details wrong.' %}

{% trans 'If this happens, SSCL will cancel the disbursement and notify the prisoner by email.' %}

{% trans 'If a bank transfer fails' %}

{% trans 'A bank transfer fails when the recipient’s account details are wrong.' %} {% trans 'If it’s sort code, SSCL identifies the issue, reports it same day, then rejects it.'%} {% trans 'If the account number is invalid, this only gets checked at the bank, then rejected.'%} {% trans 'It can take up to 10 days for the rejection notification to reach the prisoner money team even though SSCL will cancel the payment as soon as possible.' %} {% trans 'The team contacts the prison and the prison decides if a replacement cheque should be sent.' %} {% trans 'In this instant, the invoice number is no longer valid as it has been accounted for on SOP.'%} {% trans 'A new disbursement must be submitted with a new invoice number.' %}

{% random_string as content_id %}

{% trans 'How a bank transfer fails' %}

{% trans 'To cancel a cheque or bank transfer, the prison needs to raise a Service Request (SR) within 24 hours of the disbursement being sent.' %} {% trans 'It needs to follow up with a phone call to the customer service contact centre.'%} {% trans 'SSCL has 5 working days to look into the SR but they can’t guarentee a cancellation.'%} {% trans 'Once the bank is notified, it takes up to 2 days for them to cancel a cheque up to which time the cheque can still be cashed.' %} {% trans 'When a cheque is cancelled, the money moves back into SOP where it’s recorded.' %}
